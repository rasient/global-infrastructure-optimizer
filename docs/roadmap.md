# Roadmap

## v1 — MVP

- Streamlit app
- sample city profiles
- mock analysis mode
- transport scoring
- Markdown report export

## v2 — Real Map Data

- Google Maps Geocoding API
- Google Routes API
- travel time comparisons
- origin/destination route summaries
- route matrix support

## v3 — Infrastructure Layers

- public transport layer
- cycling layer
- walking access layer
- road congestion layer
- transfer hub analysis

## v4 — AI-Assisted Analysis

- OpenAI structured recommendations
- scenario comparison
- trade-off scoring
- policy brief generation

## v5 — Global Comparison

- compare multiple cities
- identify repeated system patterns
- generate global infrastructure insights
- rank intervention types by context

## Long-Term Vision

A decision-support system that helps policymakers, analysts, planners, and researchers understand mobility systems as interacting portfolios rather than isolated transport modes.
